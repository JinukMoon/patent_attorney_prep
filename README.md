# ⚖️ 상표법 암기카드

변리사 2차 시험 상표법 플래시카드 웹앱

**https://jinukmoon.github.io/patent_attorney_prep/**

## 사용법

1. **조문번호**가 표시됩니다
2. **힌트 보기** → 두문자 암기법 확인
3. **정답 보기** → 전체 문장 확인
4. **알겠음 / 모르겠음** → 자기평가 후 다음 카드로

## 기능

- **모드 선택**: 전체 / 조문(간략) / 두문자 / 문장
- **오답 반복**: 틀린 카드만 필터링
- **셔플**: 랜덤 순서 학습
- **진행률 저장**: localStorage 기반, 브라우저 종료 후에도 유지
- **PWA**: 홈 화면에 추가하면 앱처럼 사용 (오프라인 지원)
- **키보드**: `Space`/`→` 다음 단계, `←` 이전, `1` 모르겠음, `2` 알겠음
- **스와이프**: 좌우 스와이프로 카드 이동
